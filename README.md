# Sibra SaaS - Ecosystem Platform

[**Sibra SaaS**](https://saas.sibra.store) adalah platform ekosistem modern yang menyediakan berbagai tools dan solusi digital siap pakai untuk UMKM, mahasiswa, dan teknisi digital. Platform ini dirancang dengan antarmuka yang bersih, cepat, dan sepenuhnya responsif.

## Main Products

### 1. [QR Code Generator](https://saas.sibra.store/qr-code-generator/)
- Generator QR Code gratis tanpa login.
- Support **Logo di tengah** untuk branding profesional.
- Mendukung tipe data: **URL, Teks Bebas, dan WhatsApp Chat**.
- Fitur **Donasi QRIS** terintegrasi.
- Optimasi SEO untuk pasar Indonesia.

### 2. Coming Soon: Link Shortener
- Pendekkan URL panjangmu dengan analitik real-time dan kustomisasi brand yang elegan.
- **Status**: Sedang Pengembangan

### 3. Coming Soon: AI Content Writer
- Buat konten promosi dan caption media sosial secara otomatis menggunakan kecerdasan buatan.
- **Status**: Sedang Pengembangan

## Tech Stack
- **HTML5 & Vanilla JavaScript**: Menjamin performa super cepat tanpa overhead framework berat.
- **Tailwind CSS**: Desain modern, elegan, dan profesional (UI/UX fokus).
- **QRCode.js**: Library handal untuk generasi QR Code di sisi klien.

## Project Structure
- `index.html`: Landing page utama Sibra SaaS.
- `qr-code-generator/`: Folder aplikasi QR Code Generator.
- `assets/`: Folder untuk gambar, favicon, dan resource statis lainnya.
